v5.0 Modifications:

  • Starting a new fence now requires to click "Create a new fence".
  • All drawed fenced that are not being edited are red.
  • Editing fence is blue.
  • Click on a fence to edit it. Mouse over and mouse out was creating a bug.

v4.0 Modifications:

  • Multiple fences can be loaded, separated by line break;
  • Mouse over on a fence to edit, mouse out to stop editing.

v3.0 Modifications:

  • New layout.
  • Mouse scroll can now zoom on map.
  • Created button Select All to select all the text in the Points box.
  • Created button "Text 2 Map" that redraw the fence from the points in the box.
  • Created button "Draw" if the draw point is not linked to the mouse anymore, you can click to link again.
  • Geocoding session, permitting to add points in the fence with the address.
